# CITADEL-SC-DL-NLP

## Install `pipenv`

```sh
pip install pipenv
```

## Install packages for NLP

```sh
cd NLP
pipenv install
pipenv shell
python download.py # For downloading the cammembert model, nltk data, ... locally
```

## Install packages for DL

```sh
cd DL
pipenv install
pipenv shell
python download.py # For downloading .... locally
```
